Pros
No nav change. Alex is always visible → reinforces the "assistant is watching" metaphor. Chat + Insights + Settings tabs unify the full AI surface in one spot.
Cons
Drawer is narrow (~320px) → cards are cramped, descriptions truncated. Users already dismiss Alex drawer; making it home of a whole feature fights that. Settings tab inside drawer is awkward.
Scope
Extend the existing Alex recon-panel with tabs. Reuse compact insight card styling. Keep drawer dismissible.

My pick: Option A — top-level Alex Insights tab

It's the only one where the feature has room to grow. Behavioral cards need breathing room (4 across, not 2), Patterns needs its own canvas for charts, Decision History is a scrollable log. Option B ships fastest but boxes the feature into a Home widget forever; Option C buries a meaty feature in a dismissible drawer. If scope is tight, the right compromise is ship Option A's Overview sub-tab only — same scaffolding, narrower content.
